CHEMICAL ELEMENTS WEBSITE
=========================

INTRODUCTION
============
This is the Website that helps people buy Elements they need,
this could be for schools or chemists.

HOW TO EXECUTE
==============
First unzip the file and then go to the directory in your cmd terminal and install the node
modules using npm init. Start server using node server.js and visit http://127.0.0.1:8080/
in a browser. Website has been tested on multiple browsers such as Chrome and edge.
